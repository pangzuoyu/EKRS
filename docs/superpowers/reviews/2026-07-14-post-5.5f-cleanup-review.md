## Findings

### [P3] (confidence 8/10) `audit.py:78` — `_current_offset` check order is now suboptimal but correct

```python
if h is None or h.stream is None or h.stream.closed:
    return 0
```

After `close()`, `RotatingFileHandler.stream` is `None`. The previous code (`h.stream.closed`) crashed with `AttributeError`. The fix adds a `None` check before `.closed`. Correct.

Minor: order is good (None before .closed); could short-circuit further by treating `None` and `closed` interchangeably via `not h.stream or h.stream.closed`, but the current form is explicit and matches the new defensive test surface. Not worth changing.
